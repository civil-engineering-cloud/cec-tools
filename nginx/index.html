CEC Tools

服务器翻墙工具一键安装脚本，支持 V2Ray 和 Clash Meta。

功能特性

  • V2Ray: VMess 协议，KCP 传输，智能分流
  • Clash Meta: 支持 TUN 透明代理，智能分流
  • 自动安装依赖: unzip、wget 等
  • 预设节点配置: 开箱即用

快速开始

一键安装 V2Ray（推荐）

curl -fsSL http://tools.cec.cc:8410/tools/v2ray/install.sh | bash

一键安装 Clash Meta

curl -fsSL http://tools.cec.cc:8410/tools/clash/install.sh | bash

使用场景

场景 1: 临时翻墙（单次使用）

适用：临时需要访问 Google、GitHub 等

# 安装后开启代理
v2rayc on

# 当前终端可访问外网
curl https://www.google.com
git clone https://github.com/xxx/xxx.git

# 使用完后关闭
v2rayc off

注意: v2rayc on 仅在当前终端生效

场景 2: Docker 构建翻墙

适用：构建时拉取海外镜像或 npm 包

# 先开启代理
v2rayc on

# 然后正常构建
docker build -t myapp .

# 或使用代理构建
export http_proxy=http://127.0.0.1:1081
export https_proxy=http://127.0.0.1:1081
docker build -t myapp .

场景 3: 持久化代理（永久开启）

适用：需要长期翻墙服务

# 启动 V2Ray
v2rayc start

# 验证
v2rayc status
curl -x http://127.0.0.1:1081 https://www.google.com

场景 4: Clash Meta 透明代理

适用：需要全局代理，容器自动跟随

# 安装
curl -fsSL http://tools.cec.cc:8410/tools/clash/install.sh | bash

# 启动透明代理
clashc start

# 所有流量自动翻墙，无需配置
curl https://www.google.com
docker run --rm alpine/curl curl -I https://www.google.com

# 停止
clashc stop

注意: 透明代理需要 TUN/TAP 支持，Docker 容器内不支持

命令参考

V2Ray 命令

v2rayc start     # 启动 V2Ray
v2rayc stop      # 停止 V2Ray
v2rayc status    # 查看状态
v2rayc test      # 测试连接
v2rayc on        # 开启 HTTP 代理（当前终端）
v2rayc off       # 关闭 HTTP 代理
v2rayc config    # 编辑配置
v2rayc uninstall # 卸载

Clash 命令

clashc start     # 启动 Clash
clashc stop      # 停止 Clash
clashc status    # 查看状态
clashc test      # 测试连接
clashc uninstall # 卸载

注意事项

  1. 本工具仅供服务器内部网络加速使用
  2. 请遵守当地法律法规
  3. 默认配置已包含团队节点，开箱即用
  4. 配置文件位于 /usr/local/etc/v2ray/config.json

常见问题

Q: v2rayc start 启动失败？

检查日志：

tail -f /var/log/v2ray.log

Q: 如何更新节点配置？

v2rayc config
# 编辑 /usr/local/etc/v2ray/config.json

Q: 如何卸载？

v2rayc uninstall
# 或
clashc uninstall
